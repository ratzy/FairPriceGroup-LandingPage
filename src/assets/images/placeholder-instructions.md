# Hero Carousel Images Setup

## Required Images

The carousel is configured to display 3 slides. Please add the following images to this directory:

### 1. hero-C1.png
- **Primary hero image**
- Recommended size: 1920x600px (or 16:5 aspect ratio)
- Shows the main FairPrice gift card offering
- Example content: Shoppers in store with gift cards

### 2. hero-C2.png
- **Secondary hero image**
- Recommended size: 1920x600px (or 16:5 aspect ratio)
- Could showcase corporate gifting solutions
- Example content: Business professionals with gift cards

### 3. hero-C3.png
- **Tertiary hero image**
- Recommended size: 1920x600px (or 16:5 aspect ratio)
- Could showcase various occasions
- Example content: Gift cards for different celebrations

## Image Requirements

- **Format**: PNG or JPG
- **Size**: Optimize for web (< 500KB per image)
- **Dimensions**: 1920x600px minimum (responsive design will handle scaling)
- **Content**: Should complement the overlay text without being too busy
- **Color**: Consider that a dark blue overlay will appear on the right side

## Carousel Behavior

- **Autoplay**: 6 seconds per slide (based on e-commerce best practices)
- **Transitions**: Smooth fade between slides
- **Controls**: 
  - Navigation arrows (desktop only)
  - Dot indicators at bottom (all devices)
  - Keyboard navigation (arrow keys)
- **Accessibility**: Pause on hover/focus
