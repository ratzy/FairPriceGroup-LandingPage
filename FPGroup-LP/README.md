# FairPrice Gift Cards Landing Page

Responsive, component-driven landing page built with:

- HTML
- Tailwind CSS (CDN)
- CSS custom properties (variable-based theme tokens)
- Vanilla JavaScript modules for reusable interactions

## Project structure

```text
.
├── index.html
├── src
│   ├── css
│   │   └── styles.css
│   └── js
│       ├── app.js
│       ├── components
│       │   ├── accordion.js
│       │   ├── renderers.js
│       │   └── tabs.js
│       └── data
│           └── content.js
└── assets
    └── images
```

## Features

- Token-based theme using CSS variables
- Accessible tab system with keyboard navigation
- Reusable accordion component for FAQs
- Data-driven rendering for repeated sections
- Responsive layout matching the supplied Figma composition

## Run locally

Open `index.html` directly in a browser, or serve it with any static file server.
